Small Demo

Add the following markup to the HTML page on which you want the banner

Add this style link tag to your head

<link href="" rel="stylesheet" type="text/css">

Add this div and script tag to the bottom of your body, before the closing tag

<div id=" class=""></div>
<script type="text/javascript" src=""></script>
